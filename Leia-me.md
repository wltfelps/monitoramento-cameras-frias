# ❄️ ColdChain Monitor 1.0

Sistema de monitoramento em tempo real para automação térmica de câmeras frigoríficas e pátios de triagem, focado na cadeia de frio da fruticultura de exportação em Petrolina - PE (Vale do São Francisco).

## 🚀 Funcionalidades
- **Modo Alerta:** Alteração visual no exato instante em que uma cultura ultrapassa o limite térmico seguro.
- **Payload Leve:** Transmissão de dados via pacotes JSON estruturados para otimização de banda de rede.

## 🛠️ Tecnologias Utilizadas
- **Front-end:** HTML5, CSS3 (Dark Mode nativo) e JavaScript (Fetch API / Manipulação visual).
- **Back-end:** PHP (Simulação de Gêmeo Digital e estruturação de dados).

## 📁 Como rodar o projeto
1. Clone o repositório.
2. Certifique-se de ter um servidor local ativo (como XAMPP ou Laragon) para processar o script PHP.
3. Abra o arquivo `index.html` no navegador.
